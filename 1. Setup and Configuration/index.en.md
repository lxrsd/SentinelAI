## Workshop Requirements

:::alert{header="AWS Event Only" type="warning"}
**This workshop can only be run at AWS events with Workshop Studio accounts.** The workshop requires pre-configured infrastructure and Kiro CLI access that is only available through AWS Workshop Studio environments.

If you're not attending an AWS event, this workshop is not currently available for self-service deployment.
:::

## Preparing for the Workshop

This workshop requires access to AWS Workshop Studio accounts provided at AWS events. Follow the setup instructions below to prepare your environment:

**Step 1**: [Access your Workshop Studio account](01_at_aws_event) - Get your temporary AWS account credentials  
**Step 2**: [Configure Kiro CLI](02_Kiro_CLI) - Set up AI-powered security analysis tools

:::alert{header="Important" type="info"}
All workshop infrastructure is pre-deployed in your Workshop Studio account. You do not need to deploy any CloudFormation templates or create AWS resources manually.
:::
